Course Website

Coding, UI Design

As part of our Design Engineering course, I developed a fully responsive team website using HTML, CSS, and JavaScript to showcase our class projects, final term project, and team member profiles. The site features a dynamic landing page with a full-screen background image, and an intuitive navigation menu, all optimized for seamless viewing on both desktop and mobile devices.

Tools: HTML, CSS, JavaScript, Github, Photoshop

Challenges & Learnings
Balancing aesthetic appeal with performance was key; I optimized images and minimized CSS/JS files to achieve fast load times. This project strengthened my front-end development skills, teamwork documentation, and ability to create user-friendly educational platforms.
